# Diabetes Prediction Web App

A Machine Learning web application built using Streamlit to predict diabetes based on health parameters.

## Features

- Interactive Streamlit UI
- ML based diabetes prediction
- Real-time results

##model
The model is stored using pickle as trained_model.sav.

## Technologies Used

- Python
- NumPy
- Scikit-Learn
- Streamlit

## Run Locally

Install requirements


how to run:
pip install -r requirements.txt
streamlit run app.py
